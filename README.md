<h1 align="center">LabControl</h1>

## Описание
Приложение для сбора, систематизации, хранения и анализа данных получаемых в процессе производства асфальтобетонных смесей в дорожно-строительной организации. 
## Бизнес модель
Структура производства
Дорожно-строительная организация имеет несколько филиалов в регионе. В каждом филиале эксплуатируются асфальтосмесительные установки. Выпуск асфальтобетонной смеси возможен, после разработки состава в центральной лаборатории. 
Данные составы вносятся в систему. 
На асфальтобетонном заводе, производство смеси начинается с выбора доступного состава и открытия партии продукции, таким образом партия связывается с типом и видом смеси. 
По завершению сменного выпуска руководитель производства, редактирует сведния о партии - вносит количество произведенной продукции. Партия закрвается и становится недоступной для редактирования на следующие сутки в 10:00.
После открытия партии и начала выпуска, лаборатория отбирает пробу и начинает испытания, вносит результаты с привязкой к конкретной партии.
Приложение позволяет получать оперативные данные о выпуске, партии и количество, с фильтрацией по типам и видам асфальтобетонной смеси.
Есть возможность просмотра и анализа результатов испытания каждой партии.
